# Inkplate 10 Open Source Hardware files

![](https://www.crowdsupply.com/img/053e/front-white_jpg_project-body.jpg)

Open Source Hardware (OSH) files for e-paper display Inkplate 10 can be found in this repo. Inkplate 10 is a powerful, Wi-Fi enabled ESP32 based nine-point-seven-inch e-paper display â€“ recycled from a Kindle e-reader. Its main feature is simplicity. Just plug in a USB cable, open Arduino IDE/micropython/ESP-IDF, and change the contents of the screen with few lines of code. Learn more about Inkplate 10 on [Crowd Supply](https://www.crowdsupply.com/e-radionica/inkplate-10) or [official website](https://inkplate.io).
